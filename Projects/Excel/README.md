# 📊 Excel & Data Fudamentals
### 📊 Sales Analysis with Excel

## Overview

This project demonstrates the use of Microsoft Excel to clean, organise, analyse, and visualise sales data. Using PivotTables, PivotCharts, and interactive slicers, I transformed raw sales data into a dynamic report that highlights sales performance across different countries, age groups, and customer genders.

The project showcases core Excel data analysis techniques used to summarise large datasets, identify trends, and present business insights in a clear and interactive format.

## Objectives

The aim of this project was to:

- Analyse sales performance across multiple countries
- Compare customer demographics by age group and gender
- Create an interactive report using Excel's built-in analytical tools
- Present key findings through clear visualisations

---

## Data Preparation

Before building the report, I prepared the dataset by:

- Reviewing the data for consistency
- Formatting fields for accurate analysis
- Structuring the dataset for PivotTable reporting
- Ensuring the data was ready for visualisation

## Dashboard Features

The dashboard includes:

- PivotTable summarising sales by Country, Age Group and Gender
- PivotChart visualising customer demographics
- Interactive slicers for Age Group and Gender
- Dynamic filtering to explore different customer segments
- Structured report layout for easy interpretation

---

## Key Insights

The analysis identified several trends within the dataset:

- The USA generated the highest total sales.
- Adult customers accounted for the largest share of sales across most countries.
- Female customers recorded slightly higher overall sales than male customers.
- Sales patterns varied between countries, highlighting differences in customer demographics.
- Interactive slicers allow users to quickly filter the report and compare different customer groups.
